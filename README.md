# 🪽 HeronPanel

**Support:** Codesandbox, Github, Real VPS

---

**Requirement:-** `wget https://github.com/NotPhoenix07/HeronPanel/raw/main/HeronPanel.zip`

**1:-** `unzip HeronPanel.zip`

**2:-** `cd HeronPanel`

# Start Panel
**1:-** `cp .env.example .env`

**2:-** `docker compose up -d --build`

# Or on Ubuntu/Debian:

**1:-** `bash install.sh`
